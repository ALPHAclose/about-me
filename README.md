# Neural Interface Portfolio

A high-end "Neural Interface" digital OS portfolio. This project transforms a standard personal website into an immersive, premium system featuring a custom boot screen, interactive modules, glassmorphism UI, and terminal-like elements.

## Design & Aesthetics
This project uses a custom "Tech-Noir" design system:
- **Obsidian Surfaces**: Deep, warm-toned dark backgrounds.
- **Vibrant Orange Illumination**: Primary and secondary orange highlights mimicking neon or plasma.
- **Glassmorphism**: High blur, low opacity fill with thin border highlights to create depth without traditional shadows.
- **Dynamic Animations**: Smooth transitions simulating hardware powering up.

## Typography
- **Headlines**: Space Grotesk (geometric sans-serif with glow effects)
- **Body**: Manrope (breathable, human-centric sans-serif)

## Features
- **Boot Sequence**: Terminal-like initialization screen.
- **Modular Dashboard**: Content grouped into translucent, interactive pods.
- **Micro-animations**: Hover states that increase glow and blur for interactive elements.

## Running Locally
You can view the site by running a local web server in the project directory, for example:
```bash
npx serve .
```
Or simply open `code.html` in your modern web browser.
